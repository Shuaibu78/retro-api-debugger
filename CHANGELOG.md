### Keyboard Shortcuts

- `Ctrl/Cmd + Enter`: Send request
- `Ctrl/Cmd + T`: Toggle terminal
- `Ctrl/Cmd + M`: Toggle monitor
- `Ctrl/Cmd + H`: Toggle theme switcher
- `Ctrl/Cmd + R`: Toggle templates
- `Ctrl/Cmd + G`: Toggle code generator
- `Ctrl/Cmd + Shift + K`: Clear history
- `Escape`: Close panels
